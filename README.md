## 🚀 Running the project

### Mobile

Clone the project

```bash
  git clone https://github.com/VagnerNerves/boardingpass-boracodar.git
```

Enter the project directory

```bash
  cd boardingpass-boracodar
```

Install with dependencies

```bash
  npm i
```

Start the server

```bash
  npx expo start
```
